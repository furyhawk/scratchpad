This file is to help you use this routine.

1.Basic Information:
ESP32-S3-GEEK routines have specific steps and implementation methods in the information details 
of the official website wiki, and it is recommended to refer to the information before use
Official wiki:https://www.waveshare.net/wiki/ESP32-S3-GEEK

2.Notes for burning:
① When burning, you can long press the Boot button to insert USB and then release the button, 
ESP32-S3-GEEK into the Download mode to start burning, after burning and reinserting USB, 
you can enter the SPI FAST FLASH BOOT mode and run the program.
② Can turn off the Bluetooth switch of the PC, you can directly burn to run
(Bluetooth driver and ESP32-S3-GEEK COM driver conflict caused by)
