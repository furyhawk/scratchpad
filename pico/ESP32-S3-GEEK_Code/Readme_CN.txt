这个文件是帮助您使用本例程。

1.基本信息：
ESP32-S3-GEEK例程均在官网wiki的资料详情中有具体步骤与实现方法，使用前建议参照资料
官方wiki：https://www.waveshare.net/wiki/ESP32-S3-GEEK

2.烧录注意事项：
①烧录时可长按Boot按键插入USB后再松开按键，ESP32-S3-GEEK进去Download mode才开始烧录，
烧录完重新插拔USB则可以进入SPI_FAST_FLASH_BOOT mode,运行程序。
②可关闭PC的蓝牙开关，则可以直接烧录运行(蓝牙驱动与ESP32-S3-GEEK的COM驱动有冲突导致)

3.基本使用：
可查看https://www.waveshare.net/wiki/ESP32-S3-GEEK，了解每个例程的功能和操作步骤
